Single Function Tools
Live
Real-time Intensity Map
強震モニタのリアルタイム震度データを可視化。日本全国の観測網から届く微小な揺れから大地震まで、波の伝播をダイレクトに捉えます。
Live
Long-period Monitor
防災科研の長周期地震動モニタ（abrspmx_s）をリアルタイム表示。GIF画像から観測点ごとのRGB値を直接サンプリングし、Leafletマップ上に色付きマーカーで可視化します。
Map
Hypo Map Viewer
Leaflet.jsベースの震源地図。GSI淡色タイル＋陰影起伏オーバーレイ上に深さ・マグニチュード別のマーカーをプロット。JMA GeoJSON日次データをマルチデイ並列取得し、地図・テーブル双方向インタラクションで詳細を確認できます。
Analysis
Earthquake Viz
ETASモデル（地震誘発型余震系列）のパラメータ推定・フィッティングダッシュボード。Gutenberg-Richter b値、完全性マグニチュードMc、Omori–Utsu減衰則をインタラクティブに可視化。KS検定p値によるモデル適合評価機能付き。
Monitor
Elapsed Timer
主要地震からの経過時間をリアルタイム表示するモニタリングツール。最終大地震からの日数・時間・分・秒をカウントアップし、余震活動の時間的推移把握を支援します。
AI
Seismic Intensity AI
Cross-Attention Transformer による震度分布予測AI。震源位置・深さ・規模から全国約4,400観測点の計測震度を推定し、Leafletマップ上にJMA配色でリアルタイム描画します。1996〜2022年の気象庁データ（約170万観測）で学習。

Seismic Intensity — Result
震度分布予測AIの結果表示画面。推定震度をLeafletマップ上にJMA配色でプロットし、震度帯別観測点数・上位5観測点・クリック詳細を左サイドバーに表示します。
